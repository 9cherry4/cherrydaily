# Daily To Do - 일정 계획

할 일과 마감일을 입력하면 중요도, 마감 임박도, 가능한 시간대를 고려해 오늘의 일정을 자동으로 만듭니다. 데이터는 사용 중인 브라우저의 로컬 저장소에만 보관됩니다.

## 가장 간단한 실행 방법

1. `index.html` 파일을 더블클릭합니다.
2. 기본 브라우저에서 Daily To Do가 열리면 바로 사용합니다.

인터넷 연결이나 별도 설치는 필요하지 않습니다.

## 더 안정적인 실행 방법

컴퓨터에 Python이 설치되어 있다면 이 폴더에서 PowerShell을 열고 다음 명령을 실행합니다.

```powershell
python -m http.server 8080
```

그다음 브라우저에서 `http://localhost:8080`을 엽니다. 종료할 때는 PowerShell 창에서 `Ctrl+C`를 누릅니다.

## 파일 구조

- `index.html`: 화면 구성
- `style.css`: 색상과 레이아웃
- `app.js`: 저장, 반복, 일정 생성, 완료·미루기·드래그 기능
- `icon.svg`: 앱과 브라우저 탭 아이콘

## 일정 생성 규칙

- 기본 일과 시간은 07:00~21:00이며 설정 탭에서 오늘부터 변경할 수 있습니다.
- 일과 시간 변경 시 여유 시간은 새 일과 길이의 약 20%로 15분 단위 자동 조정됩니다.
- 컨디션 모드를 켜면 좋음 10%, 보통 20%, 피곤 30%, 생리 전 25% 비율로 여유 시간을 자동 조정합니다.
- 컨디션 모드를 끄면 여유 시간을 시간·분 드롭다운에서 직접 조정할 수 있습니다.
- 마감일이 없는 할 일도 등록할 수 있습니다.
- 새 할 일의 마감일은 오늘 날짜로 시작하며 필요하면 지울 수 있습니다.
- 예상 소요 시간은 시간과 분(0·15·30·45분) 드롭다운을 조합해 선택합니다.
- 0시간 0분은 저장할 수 없으며, 설정한 여유 시간을 제외한 하루 업무 가용 시간을 넘길 수 없습니다.
- 시작일 이전에는 일정 후보에 포함하지 않습니다.
- 고정 일정을 지정한 시각에 먼저 배치합니다.
- 고정 시작 시간은 시간·분 드롭다운으로 선택합니다.
- 여유 시간은 설정 탭에서 시간과 분 드롭다운으로 15분 단위 조정할 수 있습니다.
- 업무가 적은 날은 여유 시간을 1~2시간 구간으로 배치하고, 보통이면 30~60분, 일정이 빽빽하면 최소 15분까지 자동 분할합니다. 이어진 여유 시간은 하나의 카드로 합쳐 표시합니다.
- 마감일이 지났거나 가까운 일, 중요도가 높은 일을 먼저 배치합니다.
- 우선순위로 배치할 업무를 고른 뒤 가능한 시간대 전체에 균등한 기준점과 무작위 간격을 적용합니다. 중요하거나 마감이 임박한 일은 가중치만큼 앞쪽으로 당깁니다.
- 분할 가능한 일은 최대 60분, 최소 15분 단위로 배치하고 이어진 조각은 하나로 합쳐 표시합니다.
- 미룬 일은 오늘 일정에서 빠지고 다음 날 다시 후보가 됩니다.
- 완료한 일정은 오늘 일정표에서 다시 미완료로 되돌릴 수 있습니다.
- 일반 일정을 드래그해 순서를 바꾸면 시작·종료 시간이 자동으로 다시 계산됩니다. 고정 일정의 시간은 변경되지 않습니다.
- 업무와 여유 시간 카드를 다른 일정 사이 또는 목록 끝으로 옮기면 실제 빈 시각을 자동으로 계산해 다시 배치합니다. 긴 빈 시간 행은 일정표에 표시하지 않습니다.
- 브라우저가 제공하는 시간대를 화면에 표시하고, 해당 시간대의 현재 시각을 일정표 위의 얇은 선으로 나타냅니다. IP나 위치 정보는 조회하지 않습니다.
- 일정 카드에 마우스를 올리면 등록한 설명을 확인할 수 있습니다.
- 반복 업무는 매일, 여러 요일, 매월 지정일, 매년 지정일 또는 달력에서 선택한 여러 날짜로 설정할 수 있습니다. 달력의 년·월 제목을 누르면 원하는 년도와 월로 바로 이동합니다.
- 반복 일정은 대한민국 주요 공휴일을 제외하도록 선택할 수 있습니다.
- 전체 할 일과 완료 목록은 전체·일·주·월 단위로 조회할 수 있습니다.
- 이전 일정 탭에서 과거 일정표를 일·주·월 단위로 탐색할 수 있으며 당시 일정 데이터는 변경하지 않습니다.

## 데이터 초기화

브라우저 개발자 도구에서 로컬 저장소를 지우거나 사이트 데이터를 삭제하면 모든 할 일과 일정이 초기화됩니다. 현재 버전에는 실수 방지를 위해 화면 안의 전체 초기화 버튼을 넣지 않았습니다.
